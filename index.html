UniCon : Universal Controller


The aim of our project was to enhance and nurture the creativity of PEOPLE. In schools and colleges, where students need to learn, grasp and produce, the teaching methods are becoming rather mundane and uninteresting.All the brilliant creative ideas are passed on as too hard to implement because of the lack of a simple platform to use. We aim at providing the answer to these limitations with a device which will help people innovate using the simplest of things around them. We created UniCon: Universal Controller, with this device any conducting material (pencils lead(Graphite), Water, a Banana etc.) can be used to simulate an actual keyboard key on the computer. To achieve this we have used a signal detection mechanism followed by a PIC18F4550 microcontroller for interfacing using Universal Serial Bus(USB). For our project we limited ourselves to 8 keys. All the coding was done in C.
